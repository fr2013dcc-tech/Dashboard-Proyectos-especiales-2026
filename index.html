Paso a paso — Conectar con Apps Script:
1. Abre tu Google Sheet → Extensiones → Apps Script
2. Borra el código existente y pega este:
function doGet(){var s=SpreadsheetApp.getActiveSpreadsheet().getSheetByName("TU_HOJA");var d=s.getDataRange().getValues();return ContentService.createTextOutput(JSON.stringify(d)).setMimeType(ContentService.MimeType.JSON);}
3. Reemplaza TU_HOJA con el nombre exacto de tu hoja (ej: Datos)
4. Clic en Implementar → Nueva implementación → Aplicación web
5. En "Ejecutar como": Yo · "Quién tiene acceso": Cualquier usuario
6. Clic en Implementar · Autoriza · Copia la URL y pégala arriba
